# Kali-Services-Config
Script de automatización en Bash para la instalación, configuración y despliegue de servicios esenciales en Kali Linux: Postfix (SMTP Relay con Gmail), Servidor Web Apache2 con DocumentRoot personalizado y Servidor de Impresión CUPS con administración remota habilitada. Incluye manejo de credenciales SASL y limpieza de archivos de configuración.
